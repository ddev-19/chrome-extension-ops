# chrome-extension-ops
A chrome extension for ops which helps ops team to identify the type of users on internal tool. Brings name and type of user (Test, internal user, or normal) from google sheets

**Problem Statement** - Internal users (employees) and Developers (using Test account) used to raise a lot of requests (~35% of total requests recieved) and ops team had no way to identify the users, and had to validate each and every request just to realise later that it was a test account or internal user. This ops panel helped them identify the user who raised the request, saving 3–4 hrs/week in manual validation.
